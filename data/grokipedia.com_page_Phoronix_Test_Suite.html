Phoronix Test Suite

Phoronix Test Suite (PTS) is a free and open-source benchmark software for Linux and other operating systems. The Phoronix Test Suite, developed by Michael Larabel and Matthew Tippett, has been endorsed by sites such as Linux.com,[3] LinuxPlanet,[4] and Softpedia.[5] The software continues to be actively maintained, with the development branch at version 10.8.5 as of October 2025.

Features

Phoronix Test Suite supports over 450 test profiles and over 100 test suites. It uses an XML-based testing architecture. Tests available to use include MEncoder, FFmpeg and lm_sensors, along with OpenGL games such as Doom 3, Nexuiz, and Enemy Territory: Quake Wars, and many more. The suite integrates with OpenBenchmarking.org, launched in 2011 as the successor to the earlier Phoronix Global, where users may upload their test results and system information for sharing. By executing a single command, other users can compare their test results to a selected system in an easy-comparison mode. The Phoronix Test Suite supports automated Git bisecting on a performance basis to find performance regressions, and features statistical significance verification.[6]

Components

Phoromatic

Phoromatic is a web-based remote test management system for the Phoronix Test Suite. It allows the automatic scheduling of tests. It is aimed at enterprise use. It can manage multiple test nodes simultaneously within a test farm or distributed environment.[7]

Phoromatic Tracker

Phoromatic Tracker is an extension of Phoromatic that provides a public interface into test farms. It allows the tracking of a code-base or system for performance regressions, such as in the Linux kernel.[8]

PTS Desktop Live

PTS Desktop Live was a stripped-down x86-64 Linux distribution that included various versions of the Phoronix Test Suite, up to at least version 5.0. It was designed for testing and benchmarking computers from a LiveDVD or LiveUSB environment but has since been discontinued.[9]

Phodevi

Phodevi (Phoronix Device Interface) is a library that provides a clean, stable, platform-independent API for accessing software and hardware information.

PCQS

Phoronix Certification & Qualification Suite (PCQS) was a reference specification for the Phoronix Test Suite introduced in 2008.[10]

Phoronix is a technology website that offers information on the development of the Linux kernel, product reviews, interviews, and news regarding free and open-source software by monitoring the Linux kernel mailing list and conducting interviews. Phoronix was started in June 2004 by Michael Larabel, who serves as the owner and editor-in-chief.

History

Founded on June 5, 2004,[11] Phoronix started as a website with a handful of hardware reviews and guides,[12][13] moving to articles covering operating systems based on Linux and open-source software such as Ubuntu, Fedora, SUSE,[14] and Mozilla (Firefox/Thunderbird) around the start of 2005.[15] Phoronix focuses on benchmarking hardware running Linux, with a slant toward graphics articles that monitor and compare free and open-source graphics device drivers and Mesa 3D with AMD's and Nvidia's proprietary graphics device drivers. In June 2006, the website added forums to accompany news content.[16] On April 20, 2007, Phoronix redesigned its website and began publishing Solaris hardware reviews and news in addition to Linux content.[17][18] Phoronix celebrated its 20th anniversary in 2024 and continues to publish daily news, reviews, and benchmarks related to Linux hardware and open-source software as of 2025.[19] Other technical publications, such as CNET News, have cited Phoronix benchmarks.[20][21]

Open Benchmarking

OpenBenchmarking.org is a web-based service created to work with the Phoronix Test Suite. It is a collaborative platform that allows users to share their hardware and software benchmarks through an organized online interface.[22] It is primarily used for performance benchmarking and testing hardware/software performance, typically in the context of Linux-based systems.

Release history

On June 5, 2008, Phoronix Test Suite 1.0 was released under the codename Trondheim.[23] This 1.0 release was made up of 57 test profiles and 23 test suites.[24] On September 3, 2008, Phoronix Test Suite 1.2 was released with support for the OpenSolaris operating system,[25] a module framework accompanied by tests focusing upon new areas,[26] and new test profiles. On April 6, 2009, Phoronix Test Suite 1.8 was released, including a graphical user interface (GUI) using GTK+ written using the PHP-GTK bindings.[27] On September 8, 2011, Phoronix Test Suite 3.4 "Lillesand" was released, including the MATISK benchmarking module and initial support for the GNU Hurd.[28] Subsequent major releases continued to add features, support for new hardware and operating systems, and improvements to the benchmarking framework. Notable versions include:
  • 5.0 (2013): Enhanced OpenBenchmarking.org integration and mobile testing support.
  • 8.0 (2018): Simplified test creation and PHP 7 compatibility.
  • 9.0 (2019): Improved Windows support and AI/ML benchmarking tests.
  • 10.0 (2020): Support for ARM64 and RISC-V architectures.
  • 10.8.4 (July 3, 2022): Latest stable release with fixes for logging, dependencies, and hardware detection; no further major releases as of October 2025.[29]
For a full changelog, see the official GitHub repository.[1]
